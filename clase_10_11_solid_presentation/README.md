# Principios SOLID en Java - Presentación

Presentación interactiva sobre los 5 principios SOLID usando Reveal.js.

## Ver la presentación

### Opción 1: Servidor local
```bash
cd enyoi_java_slides
python3 -m http.server 8000
```
Luego abre: http://localhost:8000/clase_10_11_solid_presentation/

### Opción 2: Live Server (VS Code)
1. Instala la extensión "Live Server"
2. Click derecho en `index.html` → "Open with Live Server"

## Navegación

- `→` `←` Slides horizontales
- `↓` `↑` Slides verticales (sub-secciones)
- `ESC` Vista general
- `S` Notas del presentador
- `F` Pantalla completa
- `O` Vista de cuadrícula

## Contenido

1. ¿Qué es SOLID?
2. **S** - Single Responsibility Principle
3. **O** - Open/Closed Principle
4. **L** - Liskov Substitution Principle
5. **I** - Interface Segregation Principle
6. **D** - Dependency Inversion Principle
7. Arquitectura SOLID en acción
8. Ejercicios

## Tecnologías

- [Reveal.js 5.1.0](https://revealjs.com/)
- Tema Dracula
- Syntax highlighting con Monokai
